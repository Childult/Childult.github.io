Ubuntu踩坑系列 | 安装

Posted on 2018-03-14 | In Coding

下载iso文件

如果官网下载较慢可以选择国内的镜像：
清华大学开源软件镜像站
USTC Mirror
类似还有很多，不一一列举了。

制作安装U盘

UltraISO就可以，可以免费试用。

  1. 安装完成后打开UltraISO
  2. 文件->打开，找到下载好的iso文件
  3. 启动->写入硬盘映像…
  4. 点击写入

制作的同时会把U盘格式化，记得备份重要文件。
如果安装完成后需要重新格式化U盘，在磁盘管理选项中右键盘符选择格式化。注意要选择fat格式！！！

划分硬盘空间

进入磁盘管理，在需要划分出来的盘符上右键选择压缩卷，大小看自己需要定。成功后该段空间将变为黑色，状态变为未分配

安装

  1. 进入BIOS关闭快速启动(Fast Boot)。这一步各品牌的电脑具体操作不一样，请自行百度。
  2. 开机狂按F9进入启动顺序菜单（惠普是F9，其他品牌不清楚，同样需要自行百度）
  3. 进入安装菜单，选择Install Ubuntu或try Ubuntu before installing均可。
  4. 安装前的各种设置，看个人需要自行选择。在安装选项页面选第一项与其他系统共存，有经验可以选择最后一项自己定。

安装成功

完毕后点现在重启，重启后进入grub菜单，默认第一项是Ubuntu。
Enjoy your trip:)
